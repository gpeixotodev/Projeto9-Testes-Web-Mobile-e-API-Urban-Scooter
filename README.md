Descrição: Projeto completo de Quality Assurance envolvendo planejamento e execução de testes manuais para aplicação web e APIs. Desenvolvimento de casos de teste baseados em classes de equivalência e análise de valores limite, validação de regras de negócio, documentação de defeitos e acompanhamento dos resultados dos testes. Os cenários abrangeram funcionalidades web, dispositivos móveis e endpoints REST. Os arquivos do projeto mostram casos de teste para Web, API e validações funcionais estruturadas.

Tecnologias/Ferramentas:

Python, Selenium WebDriver, Pytest, Postman, API REST, JSON, Jira, SQL Linux (Cygwin/Bash) Git
